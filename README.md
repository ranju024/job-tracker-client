# Job Tracker Client

React frontend for the Job Application Tracker.

## Tech Stack

- React + Vite
- Material UI
- Axios
- React Router

## Features

- User registration and login
- JWT authentication with protected routes
- Job applications list with status filtering
- Add, edit, delete job applications
- Dashboard with stats

## Setup

1. Clone the repo
2. Install dependencies: `npm install`
3. Create `.env` file: `VITE_API_URL=http://127.0.0.1:8000`
4. Start the app: `npm run dev`

## Backend

Django REST API repo: [job-tracker-api](https://github.com/ranju024/job-tracker-api)
